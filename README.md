## Tecnologia
- Java

# Como executa o projeto:
Pré-requisitos: Java 17

```bash
# clonar repositório
git clone = https://github.com/fabricioddsilva/jogo-xadrez

# ir até a pasta
cd out/production/Xadrez

# executar o projeto
java app/Main
```
